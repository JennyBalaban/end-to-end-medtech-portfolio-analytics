# KPI Dictionary

## Executive Overview
- Net Revenue = sum of Closed-Won revenue (CRM)
- Case Shipments = count of shipped cases (ERP/Ops)
- Revenue per Shipment = Net Revenue / Case Shipments
- Doctor Submitters = distinct doctors with shipments
- Utilization Rate = Case Shipments / Doctor Submitters
- Marketing ROI = (Attributed Revenue - Spend) / Spend
- Delivery Forecast Accuracy = 1 - abs(Forecast - Actual)/Actual
